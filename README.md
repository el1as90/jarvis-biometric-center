## Installation

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Allow camera and microphone permissions when prompted

## Usage

### Facial Recognition
1. Navigate to **FACIAL** panel
2. Click **START CAMERA**
3. Register new users with **REGISTER USER**
4. System will detect and identify faces in real-time

### Voice Commands
1. Go to **VOICE** panel
2. Click **START LISTENING**
3. Say commands like:
   - "Open facial panel"
   - "Show statistics"
   - "Help"

### Gesture Detection
1. Open **GESTURES** panel
2. Click **START DETECTION**
3. System detects hand gestures in real-time

### Eye Tracking
1. Navigate to **EYE TRACKING** panel
2. Click **START CALIBRATION**
3. System tracks your gaze position
